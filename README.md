# SQL Database Design & Business Analytics

## Overview

This project demonstrates the design and implementation of a relational database using MySQL. The database was created to model a university system and answer business questions using SQL.

The project showcases practical SQL skills including relational database design, querying, aggregation, filtering, and advanced SQL techniques.

---

## Technologies

- MySQL
- SQL
- MySQL Workbench

---

## Skills Demonstrated

- Relational Database Design
- Primary & Foreign Keys
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- GROUP BY
- HAVING
- Aggregate Functions
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- EXISTS / NOT EXISTS

---

## Project Workflow

1. Designed a relational database schema.
2. Created tables with primary and foreign keys.
3. Inserted sample records.
4. Developed SQL queries to answer business questions.
5. Optimized queries using advanced SQL techniques.
